Spring Framework 5.0

Spring Framework 5.0的最大特点之一是响应式编程（Reactive Programming）。 响应式编程核心功能和对响应式endpoints的支持可通过Spring Framework 5.0中获得。 重要变动如下列表所示：

常规升级
对JDK 9运行时兼容性
在Spring Framework代码中使用JDK 8特性
响应式编程支持
函数式Web框架
Jigsaw的Java模块化
对Kotlin支持
舍弃的特性
