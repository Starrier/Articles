# SpringBoot Ajax CORS

## CORS ？

跨域请求，就是说浏览器在执行脚本文件的 Ajax 请求时，脚本文件所在的服务器地址和请求的服务地址不一样。说白了就是 IP 、网络协议、端口都是一样的时候，就是同一个域，否则就是跨域。这是由于安全策略——同源策略所致。是浏览器对 JavaScript 的安全限制。是放在外网脚本恶意攻击服务器的一种措施。
